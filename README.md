# MyoTools.github.io

MyoTools 官方网站 - 致力于解决细胞专家在肌纤维等细胞表型获取的各种难题，包括分割和分类标注、训练、评估、定量等完整的生态系统。

## 项目列表

### MyoSC
通过 cellposeSAM、segformer 和 imagej 解决 mATPase 和 SDH 共染色肌纤维分割、分类及定量问题。

- **GitHub**: [https://github.com/MyoTools/MyoSC](https://github.com/MyoTools/MyoSC)

## 技术栈

- HTML5
- CSS3
- JavaScript (Canvas API)
- 响应式设计
